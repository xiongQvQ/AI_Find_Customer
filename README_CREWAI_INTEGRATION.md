## 🎯 智能体协作流程

### 5个专业智能体分工

1. **需求分析专家** 📋
   - 解析自然语言需求
   - 提取产品、规格、价格、地区信息
   - 生成结构化搜索规格

2. **搜索策略专家** 🎯
   - 基于需求生成关键词
   - 制定最优搜索策略
   - 选择搜索模式和参数

3. **搜索执行专家** 🔍
   - 执行Serper API调用
   - 获取高质量公司数据
   - 数据清洗和标准化

4. **AI评分分析师** 📊
   - 多维度智能评分
   - 匹配度分析
   - 风险识别和评估

5. **结果优化专家** ✨
   - 去重和排序
   - 分级推荐
   - 生成最终推荐列表

## 🚀 使用指南

### 快速开始
1. **启动应用**: `streamlit run streamlit_app.py`
2. **配置API**: 确保 `SERPER_API_KEY` 已配置
3. **访问功能**: 点击主页的 "AI智能搜索" 卡片
4. **描述需求**: 用自然语言描述您的采购需求
5. **查看结果**: 系统自动展示评分排序的推荐公司

### 示例搜索需求
- "我想找卖数位板的公司，要求支持4K分辨率，价格1000-3000元，深圳地区"
- "需要太阳能板供应商，功率300W以上，价格合理，华东地区优先"
- "寻找LED显示屏制造商，户外P10规格，深圳或广州的厂家"
